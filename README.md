# 蝦皮分潤後台 Dashboard

這是一個輕量的蝦皮分潤追蹤後台，用來查看每月分潤、訂單、點擊、商品排行與待處理行動。

## 目前內容

- `index.html`：後台畫面
- `styles.css`：版面與視覺樣式
- `app.js`：CSV 讀取、資料整理、指標與圖表繪製
- `dashboard_spec_for_da-bing-nai.md`：第一版規格

## 使用方式

直接開啟 `index.html` 即可使用。

畫面會嘗試讀取設定好的 Google Sheet CSV。如果試算表尚未發布到網路，或瀏覽器擋住讀取，可以使用左側的「匯入 CSV」手動載入資料。

## 資料欄位

目前 dashboard 主要讀取以下欄位：

- 日期
- 月份
- 貼文URL
- 內容類型
- 商品名稱
- 品類
- 售價
- 佣金率
- 曝光數
- 互動數
- 點擊數
- 訂單數
- 實際分潤
- CTR
- CVR
- EPC
- 狀態
- 下次行動

## 目標

第一階段目標是追蹤「每月 NT$5,000 蝦皮分潤」，先把每月結果、商品成效與下一步行動整理清楚。
