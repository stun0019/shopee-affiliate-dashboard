# 分潤工具 Dashboard 簡化規格給大冰奶

## 目標

先做「每月 NT$5,000 蝦皮分潤」的輕量 dashboard。

不要做複雜後台。這一版只需要讀最多 3 個 Google Sheet 工作表，每張表不超過 10 個欄位。

- Spreadsheet ID: `1KqTwe-hXAaQW4CHMepyje6iCyzmffppPXZ8-aGLd29U`
- 主要讀前 3 個工作表。
- 請用第一列當欄位名稱。
- 欄位請用名稱 mapping，不要硬綁欄位位置。

## Dashboard 第一版畫面

第一屏只需要 5 個數字：

1. 本月分潤
2. 距離 NT$5,000 還差多少
3. 目標達成率
4. 本月訂單
5. 本月點擊

再做 4 個區塊：

1. 每日分潤折線圖：讀成效表的 `日期`、`分潤`
2. 商品排行：依 `分潤` 加總，取前 10 名
3. 待優化清單：`下次行動` 是改文案、換商品、加碼的列
4. 商品池狀態：統計待發文、已發文、暫停各幾個

## 計算規則

- 本月：用 `日期` 判斷目前月份。
- 本月分潤：加總成效表本月的 `分潤`。
- 距離目標：`5000 - 本月分潤`，最低顯示 0。
- 達成率：`本月分潤 / 5000`。
- 本月訂單：加總成效表本月的 `訂單`。
- 本月點擊：加總成效表本月的 `點擊`。

## 讀取方式

若 Google Sheet 有發布到網路，可以用 CSV 讀取各工作表。

如果遇到權限或 CORS 問題，先用其中一種：

1. Google Sheet 發布到網路，格式選 CSV。
2. 做一個簡單後端代理讀 CSV。
3. 用 Google Sheets API，API key 放後端，不要放在公開前端。

## UI 方向

- 第一眼就看到距離 NT$5,000 還差多少。
- 不要做 landing page。
- 不需要登入系統。
- 手機版要能快速看本月分潤、今日/本週要處理什麼。
- 色彩簡單：收益用綠色，警示/待處理用橘色，背景用淺色。
